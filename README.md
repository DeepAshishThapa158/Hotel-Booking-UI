<h1>Project Description<h1>

<p>The Hotel Booking System (Frontend) is a fully responsive web application designed to simulate a hotel booking experience. It features a modern and visually appealing interface with sections like a login form, a static room information page, and an automatic sliding review carousel. This project focuses on front-end design and interactivity without backend functionality.</p>

<h1>Features

<ul>Responsive Design: The website adapts to different screen sizes (mobile, tablet, desktop).
<ul>Login Section: A simple login form for simulating user authentication (no backend functionality).
<ul>Display of static room details such as price, amenities, and room type.
<ul>Review Carousel: A dynamic, automatically sliding carousel that showcases customer reviews in a visually engaging manner. The carousel transitions between different reviews to provide a seamless browsing experience.
<ul>Appealing User Interface: A clean and modern design with smooth navigation and engaging interactions.

<h1>Technologies Used

<ul>HTML: Page structure and content.
<ul>CSS: Styling, layout, and responsive design.
<ul>JavaScript: Controls the automatic sliding of the review carousel and other interactive elements
<ul>Responsive Design: Utilizes media queries for a mobile-first, adaptive layout.
